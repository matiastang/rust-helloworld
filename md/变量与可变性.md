# 变量与可变性

## 变量

`let`声明的为不可变的变量
`let mut`声明的为可变的变量，但是类型不可变
```rs
// immutable
let a = 1

let a = 'a' // 隐藏了`a = 1`

// mutable
let mut b = 1
b = 2
// const

```
