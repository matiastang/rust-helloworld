# 函数

## 语句与表达式

Rust 是一门基于表达式（expression-based）的语言，这是一个需要理解的（不同于其他语言）重要区别。

语句（Statements）是执行一些操作但`不返回值`的指令。
表达式（Expressions）计算并产生一个值。
